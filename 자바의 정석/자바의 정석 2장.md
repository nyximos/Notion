# #2 변수 variable

### 1. 변수

하나의 값을 저장할 수 있는 기억공간

### 2. 변수의 타입 Date type

- **기본형  Primitive type**
    - boolean, char, byte, short, int, long, float, double
    - 실제 값 저장
    - 타입에 따라 크기가 다르다
    1. 논리형 **boolean** 1byte
    2. 문자형 **char** 2byte
    3. 정수형 **byte** 1byte, **short** 2byte, **int** 4byte, **long** 8byte
    4. 실수형 float, **double**

- **bit** : 가장 작은 데이터 단위
- 1 bit 로 표현할 수 있는 수 : 0 , 1
- **1 byte = 8 bit** (256개의 수 표현가능)
    - 8개의 bit중 맨 왼쪽 자리 : 부호자리

        0 : 양수    1 : 음수

    - byte로 표현할 수 있는 값의 범위

        -2^7 ~ 2^7-1

        범위에 0이 포함되어 있으므로 -1을 해준다

        음수 128개 + 양수 127개 + 0 = 256개

- **참조형 Teference type**
    - 기본형을 제외한 나머지 String, System
    - 객체의 주소값을 저장
    - 모든 참조형 변수는 4byte
    - 참조형이 가질수 있는 값의 범위
        - 0x00000000~0xffffffff
        - f는 16진수로 15를 의미
        - 하나의 f는 4bit
        - 4 * 8 = 32 bit = 4byte

## 3. 변수의 선언 방법

```java
// 타입 변수명;

int score;      // int크기인 4byte 공간 만들어진다.
score = 100;    // 오른쪽 값을 왼쪽 변수에 저장

int score = 100; // 한줄로 썼을 때

String str = new String("Hello Wolrd!");
// 참조변수는 모두 4byte

```

## 4. 명명규칙 Naming convention

1. 대소문자가 구분되며 길이 제한이 없다.

    True ≠ true

2. 예약어 Reseved word 사용 금지

    true x  True o

3. 숫자로 시작할 수 없다
4. 특수문자는 _ 와 $만 허용

### 권장사항

1. 클래스 이름의 첫 글자는 항상 대문자

    변수와 메서드 이름의 첫글자는 항상 소문자

    클래스 StringBuffer

    변수나 메서드 lastIndexOf

2. 여러 단어 이름은 단어의 첫글자를 대문자로 시작

    lastIndexOf, StringBuffer

3. 상수의 이름은 대문자로 하고 단어는 _ 로 구분한다

    PI, MAX_NUMBER

## 5. 변수, 상수, 리터럴

- **변수 variable**

    하나의 값을 저장하기 위한 공간

- **상수 constant**

    한 번만 값을 저장할 수 있는 공간

- **리터럴 literal**

    그 자체로 값을 의미하는 것

## 8. 문자와 문자열

- **char**
    - 단 하나의 문자 저장
    - 둘 이상의 문자 저장시 **에러**

    ```java
    char ch = 'A';
    char ch = 'AB'; **//에러**
    char ch = '';   **//에러 :빈문자열x**
    ```

- **String 클래스**
    - 문자열 저장

```java
String st = 'A';
String st1 = "AB"; 
String st2 = "Hello Wolrd!";
String st3 = "";
```

## 10. 형변환 Casting

- **형변환**
    - 값의 타입을 다른 타입으로 변환
    - boolean을 제외한 7개의 기본형은 서로 형변환 가능
    - 형변환 하려는 타입을 괄호 안에 적어서

        리터럴이나 변수 앞에 붙여준다.

    - **자동형변환**되는 경우에는 생략하는 것이 깔끔하다

    ```java
    float f = 1.6f;
    int i = (int)f;
    ```
